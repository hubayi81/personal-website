## 🔜 联系我模块 v2.0：数据库 + 邮件双通道（方案四）

**当前状态（方案一）：** 留言存入 `contact_messages` 表，管理员登录后台收件箱查看。

**目标：** 新留言到达时同时发邮件通知，管理员不用登录也能知道有新消息。

### 需要做的事

1. **SMTP 配置**
   - `.env` 新增 `SMTP_HOST` / `SMTP_PORT` / `SMTP_USER` / `SMTP_PASS`
   - QQ 邮箱为例：开启 SMTP → 获取授权码 → 填入 `.env`

2. **异步发邮件**
   - 新文件 `app/mailer.py`（用 `smtplib` + `email.mime` 原生库，零依赖）
   - `POST /api/contact` 写入数据库后异步调用 `send_notification()`

3. **后台收件箱优化**
   - 搜索过滤（按名字/邮箱/日期范围）
   - 分页（留言多了之后）
   - 一键全部标为已读

4. **测试验证**
   - 游客提交留言 → 检查是否收到邮件
   - 断网时邮件发送失败 → 数据库写入不受影响（异步分离）
